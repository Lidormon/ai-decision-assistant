# AI Decision Assistant

AI-powered decision analysis platform built with:

- FastAPI
- OpenAI API
- HTML/CSS/JavaScript
- Multilingual support (Hebrew/English)
- RTL/LTR support

## Features

- AI decision analysis
- Modern responsive UI
- Hebrew & English support
- Structured recommendations
- Pros / Cons / Risks analysis

## AI-Assisted Development

Parts of this project were developed using AI-assisted coding tools such as Cursor and ChatGPT.
